# iBOTS-Collaboration

This repository serves as a target to test collaborative merge request workflows.
To contribute, follow these steps:

1. Fork the repository

2. Clone your own fork

3. Add yourself to the `participants/` folder by adding a text file (e.g. `jim.txt`) with a little message. Make sure the file name is unique.

4. Push the changes to your fork

5. Create a merge request and wait for it to be merged
